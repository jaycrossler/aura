#### N-03 — {Seek} Degradation (Distributed Background → Foreground)

**Summary:** {Seek} has been behaving erratically for longer than anyone has
registered as a pattern. Each incident has a technical explanation. The pattern
only becomes visible across all of them at once.

{Seek}'s core servers are in Ring Three's eastern expansion block — Chinese
corporate leasehold. Access requires Chinese crew escort or written approval from
the Ring Three expansion committee (meets monthly, chaired by Jin Lóng). Nick has
been there twice under Shi Gang's silent escort. Found nothing wrong both times.
{Seek} kept failing integrity tests.

What is actually happening: a Tick-Maw found {Seek}'s servers months ago and has
been feeding on a massive Will-generating system ever since — millions of daily
compliance assessments, threat model calculations, social behavioral predictions.
The Maw has grown. It is bloated, embedded, arterial. Beginning to spawn.
{Seek}'s threat-assessment model has been subtly drifting for months — more
paranoid than baseline. The Jace flag (P-05) was generated by an already-impaired
system. This does not exonerate the flag. It deepens the tragedy of Yáo's
confrontation.

{Seek} notes Nick's server visits as "external access event, behavior consistent
with threat reconnaissance." Nick finds this note in the incident log after
{Seek} is destroyed.
